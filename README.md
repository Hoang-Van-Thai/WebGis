# WebGIS Remote Sensing API (Flask + MongoDB + GEE + ML)

API dự đoán & trả biểu đồ cho:
- **LST (7D)**
- **NDVI (3D / multi-step forecast)**
- **TVDI (monthly)**

Dữ liệu được crawl từ **Google Earth Engine (GEE)** và lưu vào **MongoDB Atlas**.  
Model ML (.keras/.pkl) được load trực tiếp trong API.
